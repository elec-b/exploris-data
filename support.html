Before you write

A few things that often help us reply faster:

  • Which device and OS version you are using (e.g. iPhone 14, iOS 18)
  • The language you have set in the app
  • A short description of what you were doing when the problem happened
  • A screenshot, if possible

About Exploris

Exploris is a small independent project built with care. We do not have a large support team — just one developer who reads every email personally. Please be patient, and thank you for using the app.
